How it works

1

Local Verification (on your device)

Tesseract OCR reads your cedula. InsightFace matches your face to the document photo. Both models load on-demand and are deleted from memory immediately after. Zero persistent ML processes.

2

ZK Proof Generation

A Circom circuit generates a Groth16 proof: "I know a cedula + birthdate + face key such that Poseidon(cedula, birthdate, face_key) = nullifier" ‚Äî without revealing any of those values. 564ms locally.

3

Soulprint Token (SPT) Issued

An Ed25519-signed JWT is created with your trust score, credentials, and ZK proof. No name, no cedula number, no biometric data. Stored in ~/.soulprint/token.spt. Valid 24 hours.

4

Services Verify in <50ms, Offline

Any MCP server or REST API can verify your token with 3 lines of code. No internet needed. The ZK proof mathematically proves your identity without any central authority.

Integration (3 lines)

import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";
import { soulprint } from "soulprint-mcp";

const server = new McpServer({ name: "my-server", version: "1.0" });
server.use(soulprint({ minScore: 60 }));

// Access the verified identity in any tool
server.tool("my-tool", async (args, ctx) => {
  const { nullifier, score } = ctx._soulprint;
  // nullifier = unique per human, no PII
});
import express from "express";
import { soulprint } from "soulprint-express";

const app = express();
app.use(soulprint({ minScore: 40 }));

app.get("/me", (req, res) => {
  res.json({ nullifier: req.soulprint.nullifier });
  // No PII ‚Äî just the nullifier
});
import Fastify from "fastify";
import { soulprintFastify } from "soulprint-express";

const fastify = Fastify();
await fastify.register(soulprintFastify, { minScore: 60 });

fastify.get("/me", async (req) => ({
  nullifier: req.soulprint?.nullifier
}));
import { decodeToken } from "soulprint-core";

// Token in HTTP header: X-Soulprint: <SPT>
const token = decodeToken(req.headers["x-soulprint"]);

if (!token || token.score < 60) {
  return res.status(403).json({ error: "Unverified bot" });
}

// token.nullifier = unique ID per human (no PII)
// token.score = 0-100 trust score

Security Model

üî¥ Sybil Attack

Blocked. Nullifier = biometric + document. Same face = same nullifier. Can't register twice.

üî¥ Token Forgery

Blocked. Ed25519 signature bound to DID keypair. Any modification invalidates the signature.

üî¥ Replay Attack

Blocked. 24h expiry + context_tag binds proof to a specific service, preventing cross-service replay.

üî¥ Privacy Leak

Blocked. ZK proof reveals zero data. Verifier only sees: trust score, country, credential types.

üî¥ Central Bottleneck

Blocked. No central server. Verification happens offline, locally, in <50ms.

üî¥ DID Substitution

Blocked. Ed25519 signature is over the full payload including the DID. Can't swap it.
